This is the goodmodels MAGMA library v1.17 for computing gonality preserving lifts
and zeta functions of curves over finite fields.

The code is based on the algorithms presented in the paper:

"Point counting on curves using a gonality preserving lift", https://arxiv.org/abs/1605.02162

and includes the packages pcc_p and pcc_q from

"Counting points on curves using a map to P^1", http://arxiv.org/abs/1402.6758
"Counting points on curves using a map to P^1", II, http://arxiv.org/abs/1412.7217

to do the actual point counting.

For information on how to use the code, see examples.pdf.
